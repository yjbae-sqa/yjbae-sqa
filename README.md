## 🧩 Quality Engineering

### CI/CD & Automation
- CI/CD 파이프라인과 자동화 테스트를 연동하여 배포 전 검증 프로세스 실행
- Docker 기반 테스트 환경에서 자동화 테스트 실행 및 검증 프로세스 운영

### Release Quality
- Release 전 핵심 기능 정상 동작 확인을 위한 Smoke 테스트 수행
- 배포 리스크를 줄이기 위해 품질 지표 기반 Quality Gate 기준 적용 및 운영

### QA Process & Technical Testing
- Dev 협업 기반 테스트 프로세스 개선을 통해 기능 검증 및 배포 검증 흐름 개선
- 서비스 구조와 API 호출 흐름을 기반으로 End-to-End 기능 검증 수행
- 초기 조직에서 QA 프로세스 및 테스트 운영 체계 구축 및 개선
