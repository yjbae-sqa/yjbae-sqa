## 👋 안녕하세요!
Web/App/API 기반 테스트 및 자동화 경험을 보유한 Quality Engineer 입니다.  
CI/CD 환경에서 배포 안정성을 고려한 테스트 전략과, 구조적인 품질 개선 프로세스에 관심을 두고 있습니다.  
단순 검수가 아닌, Dev와 협업하며 서비스 품질을 설계하는 역할을 지향합니다.   
품질 프로세스 개선, 자동화 전략, CI/CD 기반 협업에 대해 이야기 나누고 싶으시다면 LinkedIn으로 편하게 연락 주세요. ☕

## 🧩 Quality Engineering
- CI/CD 파이프라인 내 자동화 테스트 연계
- Release 안정성 검증 및 회귀 테스트 개선 제안
- 배포 리스크 분석 및 품질 게이트 기준 반영
- Dev 협업 기반 테스트 프로세스 개선
- Docker 기반 환경에서 자동화 테스트 실행 및 검증

## ✅ Testing
- Functional: Test Case, Test Scenarios, Test Sutie, Smoke Test, Sanity Test, Regression Test
- API: API Test, API Reference Review
- Automation: TestNG, Allure, Coverage
